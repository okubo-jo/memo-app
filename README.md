# メモ帳アプリ

![アプリ画面](memo.JPEG)

## 概要
PythonのFlaskを使用して作成したシンプルなメモ帳Webアプリです。ブラウザ上でメモを入力して保存し、保存されたメモを一覧表示することができます。

Flaskの基本的な機能（ルーティング、テンプレート、フォーム処理）を学ぶことを目的として作成しました。

## 使用技術
- Python  
- Flask  
- HTML  

## 機能
- メモの入力  
- メモの保存  
- メモ一覧表示
- メモの削除  

## 画面
メモを入力して保存すると、ページ下部にメモが一覧表示されます。
